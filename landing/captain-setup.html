3 Create a New Telegram Bot for the Captain

"The Captain needs its own Telegram bot — separate from any existing agent. This takes about 2 minutes."
  1. Open Telegram, search for @BotFather
  2. Type /newbot
  3. Name: something like "My Captain"
  4. Username: must end in bot, e.g. "myname_captain_bot"
  5. BotFather gives you a token — save it:
    Captain Bot Token: _______________

Get your Telegram User ID:

  1. Search for @userinfobot in Telegram
  2. Send /start — it replies with your numeric ID
  3. Save it:
    Telegram User ID: _______________

3 OpenRouter API Key

"We need to connect the brain. OpenRouter is like a universal adapter — one account, every AI model."
  1. Go to openrouter.ai and sign up
  2. Click Wallet/Credits → add $10
  3. Set up Auto Refill: $10 when balance drops below $2
  4. Go to Keys → Create Key → name it "Captain"
  5. Save it:

What the Captain Does

Discovery

First conversation determines your path:

  • Existing business: Captain researches your website, competitors, reviews. Comes back informed. Proposes a concrete first build.
  • New business with idea: Validates the idea with market research. Proposes an MVP.
  • New business, no idea: Asks discovery questions. Surfaces 2-3 ideas. Recommends one.

Build Loop

  • Daily progress report at 9 AM — what was built, what's next, what's needed from you.
  • Builds first, asks later. Ships ugly. Never waits for permission.

Autonomy

  • Dead Man's Switch: No response for 48 hours? Captain builds the recommended option and moves on.
  • Weekly check-in: Friday proactive suggestions for improvements.
